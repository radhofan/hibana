# Hibana Screenshot Template

Keep README screenshots in this folder. Use PNG files with these names:

| File | What to capture |
| --- | --- |
| `weather-explorer.png` | Full desktop globe view with the weather panel visible |
| `city-search.png` | Search results open above the globe |
| `location-comparison.png` | Two to four compared locations visible |
| `mobile-weather.png` | Responsive mobile weather panel |

Capture at a readable browser size, avoid private data, and replace an existing file rather than changing its name so README links remain stable.
